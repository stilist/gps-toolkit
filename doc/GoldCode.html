Example

A Gold code of size 5

  var gold_code = new GoldCode(5, [5, 2, 3, 4], [5, 3], 0b10)
  gold_code.next()
  //=> 1
  gold_code.sequence
  //=> 0b1100011111110001000111100010100
  gold_code.length
  //=> 24
